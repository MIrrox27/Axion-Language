# Синтаксис языка

## Структура программы

На данный момент можно запускать код только из 1 файла, 
поэтому точкой входа является начало файла, а точкой выхода его конец.
Как и в других ЯП, в Axion код выполняется последовательно, сверху вниз.


## Комментарии

### Однострочные

Есть 2 варианта создать комментарий:
1. При помощи символа `#` (как в Python)
2. При помощи сочетания символов `//` (как в C++, C# или Java)

### Многострочные

Пока не реализованы

## Типы данных
На данный момент существует 4 типа данных:
1. integer - целые числа
2. float - числа с точкой
3. bool - логические "Истина" и "Ложь"
4. string - строки



## Ключевые слова и переменные

В Axion название переменных должны начинаться только с буквы или нижнего подчеркивания `_` 
и в названии могут иметь только вышеперечисленные символы и цифры.

Так же запрещено называть переменные зарезервированными словами. 

Полный список всех ключевых слов:

* else

* while

* foreach

* var
* val

* True

* class
* enum
