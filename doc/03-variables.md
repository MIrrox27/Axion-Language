<div align="center">
    <h3>Переменные</h3>
</div>

<div>


В Axion есть 2 типа переменных:

    - Изменяемые

    - Неизменяемые




Для того, что бы создать изменяемую переменную, 
нужно использовать ключевое слово ```var``` после него идет имя переменной и т
очка с запятой (```;```).

Пример:

```Axion
var x; # объявляем переменную x
x = 1; # присваиваем ей значение 1
```

Либо же можно сразу присвоить значение элементу:
```Axion
var x = 1;  
```



Для создания неизменяемых нужно использовать ключевое слово 
```val``` , после него идет имя константы, значение и в конце точка с запятой (```;```).

Пример:

```Axion
val x = 1; # создаем константу с именем x и присваиваем ей значение 1
```

Точно таким же образом переменным и константам можно присвоить значения типа bool (true/false), 
string (значение указывается в кавычках), float, значение возвращаемое функцией и 
значение None


</div>
